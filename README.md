# System Requirements

- Clean Arch - DDD - Repository- UseCase- MVVM
- Unit Test - Junit - Mockk

- Android Studio
- Retrofit
- Navigation Component
- Picasso
- LiveData and ViewModel 
- Coroutines

# Features

- Get characters by: name, status or gender
- Get characters by pagination
